ABOUT
Prizm is Ethereum's First Decentralized Staking Stablecoin Network (DSSN) that allows anyone to trustlessly mint custom-pegged stablecoins on the Ethereum Blockchain. These stablecoins are guaranteed by a Proof of Reserve smart contract that can be publically verified by anyone, anywhere, anytime. Prizm allows long term investors with large portfolios to trustless earn interest on any deposit. Prizm allows stablecoin owners to pay others using custom pegged stablecoins. Prizm incentivizes long term price stability using economics.
